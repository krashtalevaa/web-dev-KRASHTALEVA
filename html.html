WhenHTML5 has become a standard?What were the main reasonsfordeveloping new standard?

a) HTML5 became a standard on October 28, 2014. The main reasons for developing this new standard were to improve the language with new features, better support for multimedia elements, and more consistent behavior across different browsers.

What is a difference between W3C HTML5 and WHATWG HTML Living Standard?

b) The main difference between W3C HTML5 and WHATWG HTML Living Standard is in their development process. W3C releases stable versions of HTML standards, while WHATWG HTML Living Standard is continuously updated and does not have fixed version numbers.

What is the minimal HTML page code?

Comment on semanticand presentationalmarkup. Why presentationalmarkup is considered deprecated now?

d) Semantic markup focuses on the meaning and structure of content, while presentational markup is concerned with the appearance of content. Presentational markup is considered deprecated because it mixes content with styling, making it harder to maintain and less accessible for different devices and users.
